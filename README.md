# Cognitive Strategies In Teaching

This repository contains code for the paper **Mentalizing and heuristics as distinct cognitive strategies in human teaching** by Harootonian, Griffiths, Niv, and Ho



It is organized into the following:

- `Exp_plots.ipynb` contains code generating the figurers in the paper.
- `Model_fitting.ipynb` fits the model outputs to the choice data.

## Set up

The project uses Python 3 
(everything works with 3.10.5).
One way to use it is by setting up a virtualenv:

```
$ virtualenv -p python3 env
$ source env/bin/activate
(env)$ env/bin/pip install -r requirements.txt
```
